# LearnFlow — Smart Study Planner

**Live Site:** [asma-umar-dev.github.io/Smart-study](https://asma-umar-dev.github.io/Smart-study/)

LearnFlow is an all-in-one study companion app that helps students plan their study time, track subjects and goals, stay focused with a Pomodoro timer, and analyze their learning progress — all in one dashboard.

## ✨ Features

- **Dashboard** — At-a-glance overview of study activity, upcoming tasks, and progress
- **Study Planner** — Plan and organize study sessions and schedules
- **Subjects** — Add and manage subjects/courses being studied
- **Goals** — Set and track study goals and milestones
- **Pomodoro Timer** — Built-in focus timer using the Pomodoro technique for productive study sessions
- **Calendar** — Visual calendar view of planned study sessions and deadlines
- **Analytics** — Track study patterns, time spent, and progress over time
- **Notes** — Take and organize study notes within the app
- **Settings** — Customize app preferences

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (single-page static app)
- Local data persistence (browser-based storage)
- Hosted via **GitHub Pages**
- Responsive, sidebar-navigation dashboard UI

## 📁 Getting Started

Clone the repo and open `index.html` directly, or serve it locally:

```bash
git clone https://github.com/asma-umar-dev/Smart-study.git
cd Smart-study
# then open index.html in your browser
# or serve locally:
npx serve .
```
